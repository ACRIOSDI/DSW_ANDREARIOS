OVI diseño de sitios web

Descripción

El diseño es fundamental para el éxito de un sitio web,
este no depende solamente de que muestre contenidos,
sino que tenga un buen diseño, es decir, no basta con
crear una página mediante HTML y CSS, sino que la
página debe resultar útil y atractiva para el usuario.
El OVI Diseños de sitios web está elaborado con el
fin de dar a conocer información básica sobre el
manejo de herramientas enfocadas al desarrollo web
como lo son GIT, GITHUB, HTML5 Y CSS3. Esto
se realizará a través de lecturas, videos y actividades
cortas y fáciles de entender.

Objetivos del OVI

● El ovi pretende dar a conocer el uso básico de git y github
● Conocer el uso de html5 y su estructura.
● Conocer el uso de css3 y sus funciones.
● Fortalecer los conocimientos obtenidos por medio de
actividades.

Objetivos del curso

● Que el estudiante se fundamente teóricamente sobre las
generalidades básicas del Diseño Web.
● Que el estudiante pueda utilizar de forma correcta los
lenguajes HTML5 y CSS3 para aplicarlos en la solución de
problemas de construcción de Sitios Web.
● Que el estudiante implemente servicios y sitios web
a través de sistemas de control de versiones.

Aprende competencias generales del diseño de sitios web

Git y GitHub

HTML
